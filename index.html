Requisitos Não Aprovados

  • 🔧 Para implementar o [RNF02] "O usuário só pode errar a senha três vêzes." Deverá também ser emplementado um requisito para desbloqueio de acesso para o caso do cliente ter perdido sua senha.
  • 🔧 Em relação ao [RNF05] "O sistema só permitirá cinco consultas." Esse requisito é inviável, pois poderá ter um impácto negativo,levando em conta que o cliente pode precisar de mais opções de produto.
